
This is a **real-time monitoring and control dashboard application** built with GritShield. It provides live telemetry visualization and administrative controls for network security operations.

---
### Administrative Actions
- **Flush Counters** - Resets telemetry counters (Operator role)
- **Emergency Shutdown** - Terminates the engine kernel (Admin role)

---

### Logged Metrics Per Check
- Latency (milliseconds)
- HTTP status code
- Status message
- Error details (if any)
- Timestamp
